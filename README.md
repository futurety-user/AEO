# AEO

Download the zip file.

1. Open the project folder in your editor.
2. Launch Claude Code.
3. Run the following prompt- "Analyze the following pages:"


```text
Example:
Analyze the following pages:

https://www.packagingprice.com/kraft-tinted-paper-shopping-bags/
https://www.packagingprice.com/corrugated/
```

The output files can be found under output/analysis
